# KDE-Plasma-Setup
Personally Customized KDE Plasma Setup <br>
```sh
sudo pacman -S --needed base-devel git &&
git clone https://aur.archlinux.org/yay-git ~/Yay &&
cd ~/Yay/yay-git &&
makepkg -si &&
cd &&
yay -S --noconfirm google-chrome
```
